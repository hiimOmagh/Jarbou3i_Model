# Changelog

## 1.0.0 — Initial public release

- Public semantic versioning starts here.
- Added the Jarbou3i mascot as the app logo, favicon, welcome-card image, Open Graph image, Apple touch icon, and PWA manifest icon.
- Added optimized runtime image assets: 512 px, 192 px, 180 px, and 32 px versions. The original 2048 px mascot remains as the editable/source asset only.
- Client-side strategic analysis workbench with structured AI prompt generation, JSON import, six-layer review, weighted scoring, contradiction analysis, scenarios, evidence, assumptions, and HTML report export.
- Arabic, English, and French UI with RTL/LTR-aware layout.
- Focused export workflow: standalone HTML report only.
- Removed user-facing JSON, Markdown, Print, and runtime self-check export actions.
- Added static release checks for duplicate IDs, JavaScript syntax, optimized asset references, manifest integrity, and absence of legacy export/self-check code.
- Added theme and language persistence for a smoother returning-user experience.
