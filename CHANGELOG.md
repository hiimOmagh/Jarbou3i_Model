# Changelog

## 1.0.0 — Initial public release

- First public GitHub/GitHub Pages release of Strategic Analysis Workbench.
- Added the bilingual client-side analysis workflow.
- Added the six-layer strategic analysis model: Interests → Actors → Tools → Narrative → Results → Feedback.
- Added structured JSON import, weighted scoring diagnostics, contradictions, scenarios, evidence, assumptions, and exports.
- Added GitHub-ready documentation, fixtures, Playwright smoke tests, and GitHub Actions CI.
