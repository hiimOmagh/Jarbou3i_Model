- `biopolitics-integrity.js`: duplicate IDs, typed references, exact 13/9/5 sets, evidence verification, quantitative metadata, future versions, and migrated drafts.

Run `npm run test:ci:no-browser` for deterministic logic checks and `npm run test:browser` for the rendered workflow.
